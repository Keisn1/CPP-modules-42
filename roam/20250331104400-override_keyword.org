from C++11 on, one has =override= to be more explicit when overrriding a function
- it means, that you're overriding a [[id:af7c9361-7820-4a43-8d6e-98baac3105ac][virtual function]] of a base class

- the simple =virtual= specifier just says that a functin *may be overridden*
